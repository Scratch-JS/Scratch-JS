Scratch-JS

What is it?

Scratch-JS is an open source library designed to help beginers transition from scratch to javascript programing. When complete, every scratch block will be converted into javascript functions allowing programers to extend the functionality of scratch, into real webpages.

How to use it

After getting everything set up, you will be able to write your first bit of scratch-js code. Here are the functions available for use:

Catagory Usage Describtion
Grid grid.sizeX The horizonal size of the grid, from the middle to edge
Grid grid.sizeY The verticle size of the grid, from middle to edge
